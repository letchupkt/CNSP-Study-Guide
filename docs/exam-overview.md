# CNSP Exam Overview

## What is CNSP?

The **Certified Network Security Practitioner (CNSP)** is an "Essentials" level certification offered by **The SecOps Group**. It tests core fundamentals of network security and does not require extensive professional penetration testing experience. Instead, it focuses on a solid understanding of how security principles are applied in real-world scenarios.

## Exam Details

| Feature | Description |
|---------|-------------|
| **Exam Name** | Certified Network Security Practitioner (CNSP) |
| **Certifying Body** | The SecOps Group |
| **Format** | Multiple Choice Questions (MCQ) |
| **Question Type** | Factual and Scenario-based |
| **Duration** | 60 minutes (1 Hour) |
| **Proctoring** | Yes, AI-proctored |
| **Testing Mode** | Online and On-Demand |
| **Book Status** | Closed-book (No internet access allowed) |

## Proctoring Requirements

The exam is **AI-proctored**, which means:

- You must have a **laptop with a working camera and microphone**
- The AI system monitors your behavior during the exam
- Any suspicious activity may result in exam termination
- Ensure you're in a quiet, well-lit environment
- No other people should be visible in the camera frame

## Testing Environment

- **Online**: Take the exam from anywhere with a stable internet connection
- **On-Demand**: Schedule and take the exam at your convenience after purchasing the voucher
- **Closed-Book**: You cannot use any reference materials, notes, or internet searches during the exam
- **Single Attempt**: Each voucher typically allows one attempt

## Exam Cost

The exam voucher pricing varies. Check the official [The SecOps Group website](https://thesecops.group) for current pricing information. Typical pricing ranges from $100-$200 USD, but promotional discounts may be available.

## Who Should Take This Exam?

This certification is ideal for:

- Entry-level security professionals
- Network administrators transitioning to security
- IT professionals seeking to validate networking and security fundamentals
- Students pursuing careers in cybersecurity
- Anyone preparing for more advanced certifications (CEH, OSCP, etc.)

## What Makes CNSP Different?

- **No Official Training Required**: Self-study based certification
- **Practical Focus**: Scenario-based questions test real-world application
- **Lifetime Validity**: Certificate doesn't expire
- **Affordable**: More cost-effective than many vendor certifications
- **Flexible**: Take it when you're ready, from anywhere

## Exam Rules & Restrictions

### ✅ Allowed
- Calculator (if needed, usually built into testing platform)
- Scratch paper and pen (may need to show to proctor)

### ❌ Not Allowed
- Internet browsing or searching
- Reference materials, books, or notes
- Communication with others
- Multiple monitors (unless approved)
- Mobile phones or other devices
- Leaving the camera view during the exam

## Technical Requirements

Before scheduling your exam, ensure you have:

- Stable internet connection (minimum 2 Mbps recommended)
- Windows, macOS, or Linux laptop/desktop
- Working webcam (720p or better recommended)
- Working microphone
- Updated web browser (Chrome or Firefox recommended)
- Quiet, private testing environment
